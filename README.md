# quize-app

quize-app is a Node-express app for dealing question and answer.

## Installation

clone the quize-app from [github](https://github.com/xetnopnawab/quize-app) and install npm.

```bash
npm install --save
run npm start
open browser and goto http://localhost:3000/
```
